# Task Management App

A modern Kanban-style task manager designed for teams to collaborate efficiently, manage projects, and track work with AI-assisted workflows.

## Features

- Designed scalable data models for projects, issues, users, and roles
- Built drag-and-drop Kanban workflow with status and priority logic
- Added AI-assisted summaries/insights and activity tracking
- Optimized pagination and filtering for smooth UX at scale

## Tech Stack

- Next.js
- TypeScript
- Prisma
- PostgreSQL
- Tailwind CSS
- shadcn/ui
- AI
